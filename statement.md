#  Problem Statement

Managing personal finances can be challenging for individuals who lack proper tracking of income, expenses, and savings goals. Many people struggle to maintain a clear record of their daily transactions, resulting in poor budgeting and inability to meet financial targets. There is a need for a simple and accessible tool that allows users to store transactions, calculate balances, and plan monthly saving goals effectively.

#  Scope of the Project

The Vityarthi Project â€“ Savings and Expense Manager provides a basic console-based system for managing monetary transactions and planning savings. The tool allows users to add transactions, view recent history, compute total available balance, and calculate monthly saving requirements to reach a specific financial goal.
The project focuses only on local run-based usage without persistent database storage, advanced analytics, or graphical UI.

# Target Users

This system is designed primarily for:

Students and young individuals who need simple budgeting tools

Beginners learning finance tracking and Python programming

Users wanting lightweight expense management without complex applications

Individuals who want to calculate savings goals periodically

#  High-level Features

Add transactions including amount and description

View all recent transactions

Calculate current balance automatically

Set a savings target and months to compute required monthly savings

Simple menu-based interface for ease of use

Error handling for invalid inputs
